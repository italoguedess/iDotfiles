* iDotfiles
  The configuration files I use ;)

** Alacritty
   A blazing fast GPU-accelerated terminal emulator.

   I simply configure it to use the ~JetBrainsMono~ nerd font.
** Bspwm
  So far the best window manager I found.
  The dotfile simply defines some window rules while the bulk of the keybindings is done through the ~sxhkd~ dotfile.

  Its dotfile is used to launch the following applications:
  - sxhkd
  - polybar
  - picom
  - feh
  - redshift

** Polybar
   A really good and easy to hack standalone bar for window manager setups.
